お問い合わせ

ご意見・ご要望・バグ報告・情報の誤り指摘を歓迎します。以下のいずれかの方法でご連絡ください。

1. サイト内の意見箱（推奨）

トップページ右上の 📮 アイコンから意見箱を開けます。カテゴリを選んで本文を入力して送信すると、GitHub Issues に自動投稿されます。

  • 🐛 バグ報告 — 動作不具合、表示崩れなど
  • ❌ 情報の誤り — マッチアップアドバイスや勝率の誤り指摘
  • 💡 機能リクエスト — ほしい機能の提案
  • 💬 その他 — 感想・その他のご意見
トップページを開く

2. GitHub Issues

GitHub アカウントをお持ちの方は、直接 Issue を立てていただくのが最速です。

→ GitHub Issues を開く

既に報告されていないか検索してから投稿いただくとスムーズです。

3. 情報の誤りを見つけたら

マッチアップアドバイスやスキル説明に誤りを見つけた場合、以下の形式で教えていただけると修正が早くなります。

【チャンピオン】ダリウス vs ジャックス TOP
【該当箇所】「ダリウスのEでジャックスの反撃を無効化」
【修正内容】ジャックスのEはCCではないので「無効化」は不正確。
        正しくは「ジャックスのEの持続時間中は攻撃を控える」

4. よくあるご質問

Q. デスクトップ版が LoL クライアントを検出しません

A. LoL クライアントを先に起動してから lolpick.jp アプリを起動してください。ヘッダー左上のインジケーターが緑になれば接続成功です。それでも検出されない場合は、アプリを管理者権限で起動するか、ファイアウォールの設定を確認してください。

Q. デスクトップ版の自動更新が動きません

A. v1.1.0 初期版はホットアップデート機能が未搭載です。一度だけ 最新版インストーラー をダウンロードして上書きインストールしてください。以降は自動更新されます。

Q. Web 版とデスクトップ版は何が違う？

A. Web 版はブラウザで動作し、チャンピオンを手動入力します。デスクトップ版は LoL クライアントと連携し、チャンプセレクト中のピックを自動検出します。詳しくは 使い方 をご覧ください。

5. 返信について

個人運営のため返信に時間がかかる場合があります。特にバグ修正や情報誤りの修正は、告知なしに対応のみ行うことがあります。GitHub Issues で状況を追跡できます。

6. お寄せいただいた情報の取り扱い

意見箱から送信された情報は GitHub Issues に公開投稿されます。個人情報や機密情報は含めないようご注意ください。詳細は プライバシーポリシー をご確認ください。
