サイト全般について

lolpick.jp は無料ですか？

はい、完全無料です。アカウント登録も不要で、誰でもすぐに使えます。運営費用は広告収入で賄っています。

利用にログインやアカウント登録は必要ですか？

不要です。ブラウザで https://lolpick.jp を開くだけで即座に使用できます。

Riot Games の公式ツールですか？

いいえ、本サイトは個人運営のファンメイドツールです。Riot Games, Inc. は本サイトを承認・支援しているわけではありません。

このツールを使うことで BAN されますか？

Web 版は完全に外部サイトでの情報参照のみなので、Riot Games の利用規約に違反しません。デスクトップ版は LoL クライアントの公開API（LCU API）を読み取るのみで、ゲーム内操作や自動化は一切行いません。Riot Games は LCU API の読み取り利用を認めており、U.GG や Porofessor など多くのサードパーティツールが同様の方式を採用しています。ただし、Riot Games のポリシーは変更される可能性があります。

ツールの使い方

どうやって分析を始めますか？

トップページでスロットをクリックしてチャンピオンを選択し、最後に「マッチアップ分析」ボタンを押します。詳細は 使い方ガイド をご覧ください。

全員のチャンピオンが揃っていなくても分析できますか？

はい、味方と敵で同じレーンが1組以上揃っていれば分析可能です。揃っていないレーンは表示されません。

ピックしたチャンピオンのレーンを変更したい

PC ではドラッグ＆ドロップ、スマホではタップ→目的のスロットタップで入れ替えできます。

敵のチャンピオンのレーンを自動で判定してくれる？

はい、ピックされたチャンピオンの統計データから最も可能性の高いレーンに自動配置します。違う場合はドラッグで修正してください。

デスクトップ版について

デスクトップ版はどこでダウンロードできますか？

GitHub Releases から最新版の lolpick-setup-X.Y.Z.exe をダウンロードできます。

Web 版とデスクトップ版の違いは何ですか？

Web 版は手動でチャンピオンを入力しますが、デスクトップ版は LoL クライアントと連携し、チャンプセレクト中のピックを自動検出します。また、デスクトップ版は味方・敵のランク情報や戦績も表示できます。

Windows 以外の OS に対応していますか？

現時点では Windows 版のみ提供しています。Mac 版・Linux 版は技術的には可能ですが、対応予定は未定です。

ウイルス警告が表示されます

自己署名アプリのため、Windows Defender SmartScreen が「認識されないアプリ」として警告を出す場合があります。「詳細情報」→「実行」で進めてください。ソースコードは GitHub で公開 しているため、心配な方は自分でビルドも可能です。

アプリのアップデート方法は？

UI・データの更新は自動配信（ホットアップデート）で、再インストール不要で次回起動時に適用されます。本体の更新が必要な場合はヘッダーに更新バナーが表示されるので、ボタンからインストーラーをダウンロードしてください。

データ・情報について

勝率データはどこから取得していますか？

op.gg の公開統計データを定期的に取得しています。独自の測定ではないため、op.gg と同じ値が表示されます。

データはどのくらいの頻度で更新されますか？

勝率データは週1〜2回、マッチアップアドバイスは随時更新しています。パッチごとの大きなバランス変動時はなるべく早く追従します。

マッチアップアドバイスは誰が書いていますか？

AI（Claude）を活用して下書きを生成し、人間が内容確認・修正を行っています。誤りを見つけた場合は意見箱からご指摘ください。

ランクティア別のデータは見られますか？

現在は EMERALD 帯以上の統合データを使用しています。ティア別フィルタは今後の対応予定です。

LoL用語の解説

「有利」「不利」の定義は？

マッチアップ勝率をベースに以下のように判定しています:
  • 🟢 有利: 勝率 50.5% 以上
  • 🟡 互角: 勝率 49.5%〜50.5%
  • 🔴 不利: 勝率 49.5% 未満
単独のレーン戦での勝率なので、実際の試合はチーム全体の構成・立ち回りでも左右されます。

「パワースパイク」とは何ですか？

特定のレベルやアイテム到達時にチャンピオンが急激に強くなるタイミングです。例えばライズは Lv6 で Ultimate 獲得により大幅に強化されます。パワースパイクを把握することで、仕掛けるべき/避けるべきタイミングが分かります。

「エンゲージ」「ディスエンゲージ」とは？

エンゲージは集団戦を能動的に仕掛ける能力、ディスエンゲージは相手のエンゲージに対抗して味方を守る能力です。レオナはエンゲージ型、ジャンナはディスエンゲージ型が代表例です。

「CC」「クラウドコントロール」とは？

スタン・スネア・スロウ・ノックアップ（空中浮遊）など、敵の行動を制限する効果の総称。CC が多いチームは集団戦で有利を取りやすくなります。

「シナジー」とは？

味方同士の相性の良さ。例えばマルファイトの R（集団スタン）とヤスオの R（空中浮遊した敵に全員突進）は強力なシナジーとして有名です。

トラブルシューティング

デスクトップ版で LoL クライアントが検出されません

以下を順番に試してください:
  • LoL クライアントを先に起動してから lolpick.jp アプリを起動
  • lolpick.jp アプリを右クリック → 「管理者として実行」
  • ウイルス対策ソフト・ファイアウォールの例外設定
  • PC を再起動してから試す

チャンプセレクトのピックが自動反映されません

ヘッダー左上の LCU インジケーターが緑（接続中）になっているか確認してください。カスタムゲームで同じレーン割当のプレイヤーが複数いる場合、片方のピックが上書きされる既知問題があります。ランク/ノーマルでは発生しません。

サイト全体が表示されない・白い画面のまま

JavaScript が無効になっている可能性があります。ブラウザ設定で JavaScript を有効にしてから再読み込みしてください。また、広告ブロッカーが UI を誤検出する場合もあります。

古い情報が表示されます

ブラウザキャッシュが残っている可能性があります。Ctrl+Shift+R（Windows）または Cmd+Shift+R（Mac）でスーパーリロードしてください。

意見箱で送信できません

一時的な障害の可能性があります。その場合は直接 GitHub Issues に投稿いただくか、しばらく時間をおいて再試行してください。
