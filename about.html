サイトについて

lolpick.jp は、League of Legends（以下 LoL）のチーム構成・マッチアップ分析を、誰でも即座に確認できる無料ツールを提供するサイトです。

コンセプト

LoL のチャンプセレクトは、相手チームの構成を見て自分の立ち回りや最適なピックを考える時間的余裕がほとんどありません。本サイトは以下を目標に作られています。

  • 速度: チャンピオンを選ぶ → 数秒で分析結果を表示
  • 正確性: 公開統計データに基づく勝率・ピック率を参照
  • 実用性: 単なる勝率表示ではなく、マッチアップごとの立ち回りアドバイスを提示
  • 無料・登録不要: 誰でもすぐに使える

主な機能

Web 版（本サイト）

  • 5レーン分の味方・敵チャンピオンを手動で入力して分析
  • マッチアップ個別の勝率・相性・立ち回りアドバイス
  • シナジー（味方構成の相性）評価
  • カウンターピック候補の提示
  • レーン別のメタチャンピオン一覧
  • パワースパイク（序盤・中盤・終盤の強さ）の可視化

デスクトップ版（Windows）

  • LoL クライアント連携: チャンプセレクト中の味方・敵のピックを自動検出して即時分析
  • マッチング成立時にウィンドウを自動で前面に表示
  • 味方・敵全員のランク情報・直近戦績を表示
  • システムトレイ常駐 + Windows 起動時の自動起動対応
  • データとアプリのホットアップデート（再インストール不要で最新化）

データソース

本サイトの情報は以下の公開データソースを元に構築されています。

ソース用途
Riot Games Data Dragonチャンピオン基本情報、スキル説明、画像
op.gg勝率、ピック率、ビルド統計
独自データマッチアップごとの立ち回りアドバイス（lane_matchups.json）

注意: 統計データは各ソースから定期的に取得・キャッシュしたものです。リアルタイムではありません。

更新頻度

  • 勝率・ピック率データ: 週1〜2回の自動取得
  • マッチアップアドバイス: 随時加筆・修正
  • 新チャンピオン対応: リリース後1週間以内
  • パッチ対応: メジャーパッチごとに統計再取得

使用技術

本サイトは以下の技術を用いて構築されています。

  • フロントエンド: 素の HTML / CSS / JavaScript（フレームワーク不使用、高速ロード重視）
  • デスクトップアプリ: Electron（Windows 向けビルド）
  • LCU 接続: Node.js 自前実装（WMIC によるプロセス検出 + HTTPS ポーリング）
  • ホスティング: GitHub Pages（静的配信）
  • データ取得: Node.js スクリプト群で定期的に統計更新
  • ソースコード: GitHub で公開

運営方針

正確性と透明性

勝率・マッチアップ情報に誤りがあった場合、読者からの指摘を受けて速やかに修正します。お気づきの点があれば お問い合わせ または GitHub Issues からご連絡ください。

AI の活用

本サイトのマッチアップアドバイス生成や文章推敲には AI を活用しています。AI 生成内容は人間が確認していますが、誤りが残る可能性があります。誤情報の指摘は歓迎します。

プライバシー

利用者の個人情報を直接収集することはありません。アカウント登録も不要です。詳細は プライバシーポリシー をご確認ください。

商用利用

本サイトは個人運営の無料ツールです。運営費用を賄うため、Google AdSense による広告を掲載しています。
