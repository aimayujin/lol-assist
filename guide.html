使い方ガイド

lolpick.jp の全機能を使いこなすためのガイドです。Web 版とデスクトップ版それぞれの操作方法、分析結果の読み取り方まで詳しく解説します。

基本の使い方（Web 版）

ステップ 1: チャンピオンを選択する

トップページには左右に「MY TEAM（味方）」と「ENEMY（敵）」の 5 レーン分のスロットがあります。各スロットをクリックするとチャンピオン選択画面が開きます。

レーンは上から順に:

  • TOP — トップレーン
  • JG — ジャングル
  • MID — ミッドレーン
  • ADC — ボットレーン（キャリー）
  • SUP — サポート

ステップ 2: 分析ボタンを押す

味方と敵のチャンピオンが少なくとも1レーンずつ埋まったら、中央の「🔥 マッチアップ分析」ボタンを押します。5秒以内に分析結果が下に表示されます。

ステップ 3: 結果を読む

分析結果には以下の情報が含まれます。

項目意味
マッチアップ勝率そのマッチアップでの勝率（op.gg 統計ベース）
有利不利判定🟢 有利 / 🟡 互角 / 🔴 不利
立ち回りアドバイスこのマッチアップ特有のテクニック、注意点
全体勝率そのチャンピオンの全マッチアップ平均勝率

高度な機能（Web 版）

ドラッグ&ドロップでレーン入替

ピック済みのチャンピオンを別のレーンにドラッグすると入れ替えできます。スマホではタップ→タップで入れ替えます。

敵ロール自動推測

敵チームにチャンピオンを入れる際、そのチャンピオンが通常どのロールで使われるかを自動で判定します。例えば「ガレン」を入れると TOP スロットに配置される確率が高くなります。

ロールが違う場合は、ドラッグで正しいレーンに移動してください。

カウンターピック候補

敵のチャンピオンを先に入れてから、自分のスロットをクリックすると「この敵に有利なチャンピオン」が優先的に表示されます。

シナジー分析

味方5人が揃うと、チーム全体のシナジー（相性）を評価します。以下の観点から:

  • CC（拘束）の量
  • エンゲージ能力（集団戦の開始役）
  • 前衛/後衛バランス
  • ダメージタイプ（物理/魔法）の偏り

パワースパイク

各チャンピオンの「強い時間帯」を可視化します。早期スパイク型と後半キャリー型のバランスを見て、ゲームプランを立てるのに役立ちます。

デスクトップ版の使い方

インストール

  1. GitHub Releases から最新の lolpick-setup-X.Y.Z.exe をダウンロード
  2. ダウンロードした exe を実行すると自動でインストール＆起動します
  3. ウイルススキャン警告が出る場合は「詳細情報」→「実行」で許可してください（自己署名アプリのため）

初回設定

起動後、特別な設定は不要です。LoL クライアントが起動していれば自動で接続します。

ヘッダー左上に「LoLクライアント接続中」と緑のドットが出れば接続成功です。

チャンプセレクト自動検出

ランク or ノーマルマッチで試合を見つけてチャンプセレクトに入ると、アプリは自動でピックを取り込みます。

  • 味方・敵のピントが確定するたびに自動で UI に反映
  • 全員ピック完了時点で自動的に分析開始
  • BAN情報も自動反映

マッチング成立時の自動前面表示

Ready Check（マッチング成立）画面が出たタイミングで、lolpick.jp のウィンドウが自動的に画面の最前面に表示されます。別アプリを操作中でも即座に分析を確認できます。

味方・敵のランク情報と戦績

チャンプセレクト中、味方と敵全員の以下が自動表示されます。

  • ソロ/デュオランクのティア・LP・勝率
  • 直近10試合の勝敗・KDA・CS

ヘッダー右上の「📜 戦績」ボタンで詳細モーダルを開けます。

システムトレイ常駐

✕ボタンを押してもアプリは完全終了せず、システムトレイ（タスクバー右下）に常駐します。LoL クライアントが起動すると自動的にウィンドウが再表示されます。

  • ダブルクリック: ウィンドウを再表示
  • 右クリックメニュー: 「Windows起動時に自動起動」の切替、「終了」

自動起動

トレイアイコンを右クリックして「Windows起動時に自動起動」にチェックを入れると、PC 起動時にアプリがトレイに常駐します。LoL クライアントを立ち上げると自動的にウィンドウが表示されます。

データ更新と自動アップデート

データの自動更新

アプリ起動から3秒後にバックグラウンドで最新データをチェックします。以下が自動的に更新されます。

  • 勝率データ（winrate_cache.json）
  • マッチアップアドバイス（lane_matchups.json）
  • 全体勝率データ
  • ビルド情報
  • メタチャンピオンリスト

アプリのホットアップデート

UI やロジックの修正は、再インストール不要で自動配信されます。アプリが version.json を見て新バージョンを検出すると、新しい index.html をダウンロードして次回起動時に適用されます。

本体アップデート（インストーラー必要）

LCU 接続方式の変更や Electron バージョンアップなど、本体の更新が必要な場合はヘッダー上部に更新バナーが表示されます。「ダウンロード」ボタンを押すと最新インストーラーが取得できます。

分析結果の読み取り方

勝率の目安

勝率判定推奨アクション
55% 以上🟢 非常に有利積極的に仕掛ける。トレード・ガンク要請
50.5% 〜 55%🟢 やや有利小有利を活かす立ち回り
49.5% 〜 50.5%🟡 互角個人技・ガンク・ドラゴンで差をつける
45% 〜 49.5%🔴 やや不利序盤は守備的に、後半に備える
45% 未満🔴 非常に不利JG に助けを呼ぶ、レーンスワップ検討

立ち回りアドバイスの活用

各マッチアップには、そのチャンピオン組み合わせ特有の立ち回り情報が記載されます。例えば:

  • 「相手の◯◯スキルクールダウン時を狙う」
  • 「レベル6までに◯◯を阻止する」
  • 「Eのアクティブ中は攻撃を控える」

短時間で読める要点のみに絞っているので、チャンプセレクト中でも確認できます。

モバイルでの使い方

Web 版はスマートフォンにも最適化されています。ブラウザで https://lolpick.jp を開いて通常通り使えます。

  • タップでチャンピオン選択
  • スクロールでピック済みキャラの確認
  • ホーム画面に追加すると PWA のように使えます

うまく動かないとき

分析ボタンを押しても結果が出ない

  • 味方と敵の両方に、同じレーンのチャンピオンが1組以上いるか確認
  • ブラウザのキャッシュをクリア（Ctrl+Shift+R）
  • JavaScript が有効になっているか確認

デスクトップ版で LoL クライアントを検出しない

  • LoL クライアントを先に起動してからアプリを起動
  • ウイルス対策ソフト・ファイアウォールの許可設定
  • アプリを管理者権限で起動してみる

チャンプセレクトのピックが反映されない

  • カスタムゲームで同じロールが2人いる場合、片方のピックが上書きされる既知の問題あり。ランク/ノーマルでは発生しません
  • ヘッダー左の LCU インジケーターが緑か確認
