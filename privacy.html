プライバシーポリシー

最終更新日: 2026年4月18日

lolpick.jp（以下「本サイト」）は、利用者のプライバシーを尊重し、以下の方針に基づき個人情報・利用データを取り扱います。

1. 収集する情報

本サイトは、利用者の個人情報を直接収集するフォームやアカウント登録機能を有していません。以下のデータが、サイト利用時に自動的に記録または処理される場合があります。

  • アクセスログ: IPアドレス、ブラウザの種類、リファラ、閲覧日時など。GitHub Pages のサーバーログとして記録されます。
  • ローカルストレージ: 言語設定・テーマ設定・入力されたチーム構成などを、利用者のブラウザ内にのみ保存します。本サイトのサーバーには送信されません。
  • Cookie: アクセス解析および広告配信の目的で、第三者サービスが Cookie を利用する場合があります（下記参照）。

2. Cookie と第三者サービス

本サイトは以下の第三者サービスを利用しています。各サービスのプライバシーポリシーもご確認ください。

2-1. Google AdSense

本サイトは広告配信に Google AdSense を使用しており、Google および広告パートナーが Cookie および識別子を使用して、利用者のこのウェブサイト及び他のウェブサイト上での利用履歴に基づき、広告を配信することがあります。

パーソナライズ広告を無効化する場合は、Google の広告設定 または aboutads.info からオプトアウトできます。

2-2. アクセス解析

本サイトではアクセス解析ツールとして Google Analytics を導入する可能性があります。Google Analytics はトラフィックデータの収集のために Cookie を使用しています。このトラフィックデータは匿名で収集されており、個人を特定するものではありません。

2-3. 外部データソース

  • Riot Games Data Dragon: チャンピオン情報・画像の取得
  • op.gg: 勝率・ピック率などの統計データ参照
  • GitHub Pages / GitHub Releases: ホスティングおよびデスクトップ版配布

3. デスクトップ版について

デスクトップ版（Windows 向け Electron アプリ）は、ローカル PC 上で以下の処理を行います。

  • LoL クライアント（LCU API）への接続 — ポート番号・認証トークンを wmic コマンド経由で取得し、ローカル内でのみ使用します。本サイトのサーバーには送信しません。
  • ランク情報・戦績の表示 — LCU API から取得した情報を画面に表示するのみで、外部に送信しません。
  • アプリ更新チェック — https://lolpick.jp/version.json および https://lolpick.jp/index.html を定期取得します。通常の HTTP アクセスログ以外の情報は送信しません。

4. データの利用目的

  • サイト利用状況の把握と改善
  • 広告配信（パーソナライズ広告を含む）
  • 不正アクセス・スパム対策

5. 第三者への提供

本サイトは、法令に基づく場合を除き、利用者の個人情報を第三者に開示・提供することはありません。

6. お問い合わせ

プライバシーポリシーに関するご質問は お問い合わせ ページからご連絡ください。

7. 免責事項

本サイトは第三者サービスから取得したデータ（Riot Games、op.gg 等）を利用していますが、その正確性・最新性について保証するものではありません。本サイトの情報を利用した結果生じた損害について、運営者は一切の責任を負いません。

8. 改定について

本プライバシーポリシーは、必要に応じて予告なく変更することがあります。変更後のポリシーは本ページに掲示された時点で効力を生じます。
